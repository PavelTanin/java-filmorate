# java-filmorate
Repository for Filmorate project.

Web приложение с информацией о кино. Пользователи имеют возможность комментировать и оценивать фильмы. В первой версии приложения хранение осуществлялось при помощи HashMap. Далее хранение было реализовано во встроенной H2 базе данных. Сам сервис реализован при помощи фреймворка Spring и сборщика Maven. Также использовались: JdbcTemplate, Lombok, написаны собственные мапперы для сущностей.
